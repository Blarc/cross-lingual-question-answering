# Getting started
To get started, fork or clone the repository.

## Repository structure
The repository contains the following folders:
- `article` contains the latex files and the article in pdf
- `data` should contain the data used for teaching the models (not commited to the repository because of the size)
- `docs` contains the files for generating documentation in [Sphinx](https://www.sphinx-doc.org/en/master/index.html)
- `src` contains the code
