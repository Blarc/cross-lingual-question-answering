API Documentation#

to_lines.py#

to_lines.to_lines(data_name, number_of_parts)#

Converts all json data, that needs to be translated to separate lines. These files with lines can be sent to eTranslation for translating. For keeping the files small enough, parameter :param number_of_parts can be specified.

Parameters

  • data_name – name of the data file (train, dev)

  • number_of_parts – number of files the translated lines are split to

translate.py#

translate.translate(data_name, data_suffix, number_of_parts)#

Translates the json data with the help of the translated lines. The function expects files with translated lines to exist in the data directory.

Parameters

  • data_name – name of the data file (train, dev)

  • number_of_parts – number of files the lines are going to be split to

translation_utils.py#

translation_utils.average_similarity(query_ngrams_embeddings, context_ngrams_embeddings)#

Calculates average similarity between the most similar query n-grams and context n-grams.

Parameters

  • query_ngrams_embeddings – n-grams for the text we want to find in context

  • context_ngrams_embeddings – n-grams for the context

Returns

average of n-grams similarities with the highest similarities

translation_utils.clean_text(text_)#

Cleans the text, so it can be used to find similar words.

Parameters

text – The text we would like to clean.

Returns

Cleaned text.

translation_utils.find_similar_text(query, context, fasttext_model, start_n=1, end_n=3)#

Finds the most similar word in text based on the FastText word embedding model and cosine similarity between n-grams.

Parameters

  • query – the text we are searching for

  • context – the text in which we are searching

  • fasttext_model – pre-trained FastText model for calculating word embeddings

  • start_n – the size of the smallest n-gram

  • end_n – the size of the larges n-gram

Returns

index of the first word in the context and the weighted similarity score

translation_utils.get_grams(text, from_=1, to_=3)#

Takes text as an input and returns N-grams, which are consecutive strings of N words.

Parameters

  • text – Text to convert to n-grams.

  • from – The size of the smallest n-gram.

  • to – The size of the biggest n-gram.

Returns

Generator that returns n-grams.

translation_utils.grams_to_embeddings(ngrams_list, model_)#

Transforms N-grams to word embeddings that are needed for calculating similarity between N-grams.

Parameters

  • ngrams_list – N-grams which we want to transform.

  • model – FastText model used for calculating word embeddings.

Returns

FastText word embeddings vectors.

translation_utils.weighted_similarity(average_similarities)#

Calculates weighted similarities based on the formula from the article. The longest n-gram has the biggest weight.

Parameters

average_similarities – average similarities between the most similar n-grams

Returns

weighted similarity of different lengths of n-grams
